# Query Results

These CSVs are generated by `tools/run_sql_analysis.py` from the public Olist source files.

- `00_load_validation.csv` records expected row counts, tested orphan counts, and the repeated-review diagnostic.
- `01_...csv` through `10_...csv` correspond directly to the numbered SQL files in `queries/`.

The raw source files are not committed. Refresh the outputs by downloading the dataset listed in the project README, placing the nine CSVs in `raw/`, and running the local analysis tool.
